PeakURL

The modern self-hosted link management platform.

PeakURL gives you branded short links, analytics, user access, and API tools from one clean dashboard. This package can live in the site root or in a subfolder.

Author: Abd Ur Rehman Stack: React admin, PHP app runtime, MySQL or MariaDB Install flow: setup-config.php then install.php

Getting Started

  1. Upload the package and extract it into an empty folder on your hosting account.
  2. Create a database and database user from your hosting panel.
  3. Open setup-config.php to enter the database details and write config.php.
  4. Continue to install.php to set the site title, admin username, email, password, and table prefix.
  5. After installation, sign in and manage your links from /dashboard.

Before You Install

Server Requirements

  • PHP 7.4 or newer
  • pdo and pdo_mysql enabled
  • MySQL 8.0 or MariaDB equivalent
  • Apache with mod_rewrite

Write Access

  • Site root must be able to write config.php
  • PHP 8.4 is recommended for current deployments
  • HTTPS is recommended for secure session cookies

What Is Included

Site Files

  • index.php boots the app and routes requests
  • setup-config.php handles database configuration
  • install.php handles site and admin setup
  • config-sample.php provides the config template

Application Files

  • assets/ contains the built admin app
  • app/ contains the runtime, API, and services

Deployment Notes

  • PeakURL runs from the installed site URL and supports site-root or subfolder deployments.
  • If you want it under a sub-path, extract it into that folder and run setup there.
  • Keep backups of your database and generated config.php before upgrades.
  • Store installation credentials and API keys securely.
